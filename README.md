# body status recording app
简单的uniapp实现,安卓端可打包安装使用和H5可转码压缩部署。  
1.轮播的欢迎页（bing 每日一图）  
2.登录  
3.身体状态信息展示  
## Installation
 1.HBuilder X 打开  
 2.$ npm i
